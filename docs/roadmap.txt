django-sophie future 1.0 Features
=================================

This is a list of features planned for a potential 1.0 release of Sophie.

*   Blog Posting
*   Category
*   Feed
*   Sitemap
*   Comments
*   Code highlighting
*   Pingback
*   Tagging
*   Wordpress import
*   Blogger import
*   Mobile friendly UI
*   Editing in markdown, reStructuredText 
    and potentially other markup languages.
*   Export blog content to some XML format
*   Creating pages
*   Be independent of templates, aka have templatetags instead of raw template
*   Links
*   Tests
*   Documentation
*   Update Pinging to service like Feedburner, etc.
